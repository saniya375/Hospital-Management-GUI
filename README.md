# हॉस्पिटल मॅनेजमेंट GUI (Hospital Management GUI)

## प्रकल्पाची माहिती
हा Java Swing वापरून बनवलेला एक डेस्कटॉप GUI ऍप्लिकेशन आहे. यामध्ये हॉस्पिटलमधील रुग्णांची माहिती एका ग्राफिकल इंटरफेसद्वारे जोडणे, बदलणे, काढून टाकणे आणि शोधणे या सुविधा आहेत. सर्व रुग्णांची माहिती टेबलमध्ये लगेच दिसते.

## वैशिष्ट्ये (Features)
- **Add Patient** – नाव, वय, लिंग, आजार, नेमलेला डॉक्टर आणि स्थिती टाकून नवीन रुग्णाची नोंद जोडता येते
- **Update Selected** – टेबलमधील एखादी नोंद निवडून तिची माहिती बदलता येते
- **Delete Selected** – निवडलेली नोंद पुष्टीकरणानंतर काढून टाकता येते
- **Search by Name** – नावावरून रुग्ण शोधता येतो
- **Show All** – सर्व रुग्णांची यादी पुन्हा दाखवता येते
- **Clear Form** – फॉर्म आणि निवड रिकामी करता येते

## रचना (Design)
- `Patient` – प्रत्येक रुग्णाची माहिती (ID, नाव, वय, लिंग, आजार, डॉक्टर, स्थिती) सांभाळतो
- `JTable` – सर्व रुग्णांची यादी दाखवतो; नोंद निवडल्यास ती फॉर्ममध्ये लोड होते
- फॉर्म पॅनल – माहिती टाकण्यासाठी आणि Add/Update/Delete/Clear बटणे
- सर्च पॅनल – नावावरून शोधण्यासाठी

## कसे चालवायचे (How to Run)
हा एक ग्राफिकल ऍप्लिकेशन आहे (कन्सोल नाही), त्यामुळे चालवल्यावर एक विंडो उघडेल:
```bash
javac HospitalManagementGUI.java
java HospitalManagementGUI
